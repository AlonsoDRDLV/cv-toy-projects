# Proyectos de Visión por Computador (OpenCV)

Colección de pequeños proyectos centrados en técnicas de visión por computador utilizando OpenCV.

---

## real_time_filters
Filtros de imagen en tiempo real: ajuste de contraste, manipulación de color, distorsión, desenfoque facial y efectos basados en fractales.

---

## edge_detection
Implementación de tres operadores de detección de bordes: Sobel, Scharr y Canny.

---

## shape_descriptor_classifier
Entrenamiento e inferencia de un modelo para clasificar formas usando descriptores de forma (área, perímetro y momentos de Hu).

---

## panoramics
Creación de panoramas a partir de conjuntos de imágenes, extrayendo puntos de interés, emparejándolos y calculando las homografías mediante RANSAC.
